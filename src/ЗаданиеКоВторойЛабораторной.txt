Требования к заданиям по тестированию алгоритмов сортировки.
1. Общие требования:
⎯ Реализовать минимум 3 алгоритма сортировки из указанных в варианте
⎯ Провести тестирование на различных типах массивов
⎯ Измерить время выполнения для каждого алгоритма
⎯ Сделать сравнительный анализ результатов
2. Тестовые массивы (обязательные для всех вариантов):
2.1. По размеру:
⎯ Малый массив
⎯ Средний массив
⎯ Большой массив
2.2. По характеристикам:
⎯ Случайный массив (равномерное распределение)
⎯ Частично отсортированный массив (75% элементов на своих местах)
⎯ Обратно отсортированный массив
⎯ Массив с множеством дубликатов (10% уникальных значений)
⎯ Почти отсортированный массив (90% элементов на своих местах)
3. Метрики для измерения:
3.1. Время выполнения:
⎯ Замерять в наносекундах/миллисекундах
⎯ Проводить минимум 5 запусков для каждого теста
⎯ Учитывать лучшее время или среднее значение

Вариант 4
Сравните пузырьковую сортировку (простая), сортировку перемешиванием
(средняя) и пирамидальную сортировку (быстрая).
