Часть 1: Односвязный список (Singly Linked List)
Требования к реализации:
Класс Node - должен содержать:
• Поле для хранения данных (типа T)
• Ссылку на следующий узел (next)
Класс SinglyLinkedList - должен реализовать методы:
1. addFirst(T data) - добавление элемента в начало списка
2. addLast(T data) - добавление элемента в конец списка
3. removeFirst() - удаление первого элемента
4. removeLast() - удаление последнего элемента
5. remove(T data) - удаление элемента по значению
6. contains(T data) - проверка наличия элемента
7. size() - возврат размера списка
8. isEmpty() - проверка на пустоту
9. display() - вывод всех элементов списка
10. clear() - очистка списка
Часть 2: Двусвязный список (Doubly Linked List)
Требования к реализации:
Класс Node - должен содержать:
• Поле для хранения данных (типа T)
• Ссылку на предыдущий узел (prev)
• Ссылку на следующий узел (next)
Класс DoublyLinkedList - должен реализовать методы:
1. Все методы из SinglyLinkedList
2. add(int index, T data) - вставка элемента по индексу
3. remove(int index) - удаление элемента по индексу
4. get(int index) - получение элемента по индексу
5. displayReverse() - вывод элементов в обратном порядке
6. getFirst() - получение первого элемента
7. getLast() - получение последнего элемента
Часть 3: Циклический список (Circular Linked List)
Требования к реализации:
• Вариант на выбор: циклический односвязный или двусвязный список
• Особенность: последний элемент ссылается на первый
Класс CircularLinkedList - должен реализовать методы:
1. Все базовые методы из SinglyLinkedList
2. rotate() - циклический сдвиг списка (первый элемент становится последним)
3. findCycle() - проверка наличия цикла (должна всегда возвращать true)
4. find(T data) - поиск элемента с учетом циклической природы
5. splitIntoTwo() - разделение списка на два равных циклических списка
