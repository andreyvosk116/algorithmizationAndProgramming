Бинарные деревья
Реализуйте структуру узла бинарного дерева на Java.
1. Реализуйте базовые методы обхода дерева:
− preOrder (прямой порядок)
− inOrder (центрированный порядок)
− postOrder (обратный порядок)
− levelOrder (поуровневый порядок)
− Реализуйте метод height для вычисления высоты дерева.
2. Полное бинарное дерево.
− Постройте полное бинарное дерево (Full Binary Tree), где каждый узел имеет
либо 0, либо 2 потомка. Минимальная высота дерева - 3.
− Реализуйте функцию isFull(), которая проверяет, является ли дерево полным.
3. Бинарное дерево поиска (BST)
− Постройте бинарное дерево поиска (Binary Search Tree) из
последовательности целых чисел.
4. Сбалансированное дерево (Ручная балансировка)
− Постройте сбалансированное бинарное дерево (разница высот левого и
правого поддеревьев любого узла ≤ 1) из отсортированного массива.
5. Реализация базовых операций
− Реализуйте бинарное дерево, используя стратегию вставки по уровню (level
order).
− Метод insert(int data) для вставки нового элемента в первую свободную
позицию при обходе в ширину.
− Метод delete(int key) для удаления узла путем замены на самый глубокий
правый узел
